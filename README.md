# RecoM3ndo Traveler Assistant

A lightweight web app that helps travelers find recommendations across:

- Dining
- Hotels
- Entertainment
- Contractors
- Government assistance
- Employment programs

## Run locally

```bash
python3 -m http.server 8000
```

Then open <http://localhost:8000>.

## How it works

Users choose destination, budget, need type, and travel style. The app scores and filters listings, then shows the top recommendations.
